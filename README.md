# patterns_js
Implementing patterns on javascript and typescript.

Patterns implemented:
 * Behaviour:
    - chain_of_responsibility
    - command
    - iterator
    - mediator
    - memento
    - observer
    - state
    - strategy
    - template
    - visitor
 * Structural
    - adapter
    - bridge
    - composite
    - decorator
    - facade
    - flyweight
    - proxy
 * Creational
    - builder
    - factory
    - prototype
    - singleton
